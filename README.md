# Age Calculator 🎂

A sleek, responsive, and visually stunning Age Calculator built with HTML, CSS, and JavaScript.

## ✨ Features

- **Modern Glassmorphism UI**: Beautiful frosted-glass effect with a colorful animated gradient background.
- **Accurate Age Calculation**: Computes your exact age down to the precise number of years, months, and days.
- **Input Validation**: Prevents users from selecting future dates to avoid incorrect calculations.
- **Clean Typography**: Uses the "Poppins" Google Font for a crisp, legible, and modern appearance.
- **Fully Responsive**: Adapts nicely to both desktop and mobile screens.

## 🚀 How to Run

1. Clone or download this repository to your local machine.
2. Open the `index.html` file in any modern web browser (Chrome, Edge, Firefox, Safari).
3. Select your birth date from the date picker and click **Calculate**!

## 🛠️ Built With

- **HTML5**: For the structure of the web page.
- **CSS3**: For the Glassmorphic layout, gradient animations, and overall styling.
- **Vanilla JavaScript**: For capturing user input and handling the complex date math logic.
